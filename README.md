# WeIP Messenger Lite

**WeIP Messenger Lite** is a lightweight local network messaging and file sharing application for Windows 10 & Windows 11.

It allows devices on the same network (Wi-Fi or Ethernet) to discover each other and exchange messages and files without requiring internet access.
